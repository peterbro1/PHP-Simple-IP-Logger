# PHP-Simple-IP-Logger
A simple PHP IP/geoip logger

#### Usage

To use this, simply add

```
<?php
require_once('log-ip.php');
log_ip('[PATH TO LOG]');
?>
```
To the beginning of your pages.

It will log it in the specified file with neat formatting:
